# airhorizon-legal

Legal documents for the AirHorizon iOS application.

## Files
- `TERMS.md` - English Terms of Use in Markdown
- `terms.html` - English HTML version for website / GitHub Pages
- `PRIVACY.md` - English Privacy Policy in Markdown
- `privacy.html` - English Privacy Policy HTML version for website / GitHub Pages

## Suggested publishing flow
1. Push this repository to GitHub.
2. Enable GitHub Pages from the repository root on the `main` branch.
3. Use the published `terms.html` and `privacy.html` URLs in App Store Connect and in the app paywall.

## Notes
- The Terms are intentionally tailored to AirHorizon's motion, heading, location, barometer, and external telemetry functionality.
- The Privacy Policy is tailored to on-device processing, StoreKit entitlements, local network telemetry, GPS/GNSS, and barometric data.
- The text includes expanded safety, non-aviation-certification, privacy, and liability limitation language.
