# airhorizon-legal

Legal documents for the AirHorizon iOS application.

## Files
- `TERMS.md` - source Terms of Use text in Markdown
- `terms.html` - publishable HTML version for website / GitHub Pages

## Suggested publishing flow
1. Push this repository to GitHub.
2. Enable GitHub Pages from the repository root on the `main` branch.
3. Use the published `terms.html` URL in App Store Connect and in the app paywall.

## Notes
- The Terms are intentionally tailored to AirHorizon's motion, heading, and location-based functionality.
- The text includes expanded safety, non-aviation-certification, and liability limitation language.
